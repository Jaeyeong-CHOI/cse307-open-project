# 2026-03-21 Research Log

## Objective
- 연속 실행 모드로 OCaml 기반 연구 도구 체인 착수
- alias 충돌 검증 자동화 최소 기능 구현

## What was changed
- `ocaml-confusion-lang/` 초기 프로젝트 생성
  - `dune-project`
  - `bin/main.ml` (validate/roundtrip CLI)
  - `examples/case-c2.tsv`
  - `README.md`
- `validate` 명령에서 아래 검증 수행:
  - 필수 키 존재(in/def/for/return/if/elif)
  - alias phrase 중복 감지(1:1 위반)

## Results / observations
- 시연 중 자주 발생하던 if/elif/for/return alias 충돌을 사전 차단할 수 있는 기반 확보
- JSON 파싱 대신 TSV 기반으로 시작하여 의존성/비용/복잡도를 낮춤 (cheap-first)

## Critical review
- 현재 `roundtrip`은 스텁 상태라 실험 자동화는 불완전
- tokenizer/replace/semantic equivalence까지 이어져야 연구 검증력이 충분해짐

## Next steps
1. tokenizer(문자열/주석 보호) 구현
2. single-pass replace 엔진 추가
3. roundtrip checker v1 구현
4. 결과를 `docs/research/results/` 스키마로 저장
