# Literature Review (Curated)

최신 연구를 무작정 쌓지 않고, **연구 질문과 직접 연결되는 것만** 선별해 정리합니다.

## 선별 기준 (Inclusion)
1. LLM 코드 생성/디버깅/학습 효과를 실험적으로 다룸
2. 비교군(도구 사용/미사용, 프롬프트 조건 등)이 명확함
3. 지표/실험 설계/한계가 문서에 명시됨
4. 본 프로젝트 질문(준수율/회귀/학습효과)에 직접 연결 가능

## 제외 기준 (Exclusion)
- 주장만 있고 실험 설계가 불명확한 글
- 동일 주장 반복 기사/2차 인용
- 우리 질문과 무관한 일반 AI 트렌드 글

## 정리 포맷
- Source
- Claim
- Evidence strength (High/Medium/Low)
- Relevance to this project
- How we integrate (실험/지표/해석에 반영 방식)
- Limitations / caveats
