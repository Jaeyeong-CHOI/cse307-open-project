# L2-v4 - Fibonacci / Syntax Inversion Language

[Language Description]
This is NOT Python. It uses inverted syntax.
ALL standard Python keywords (def, return, if, for, in) are FORBIDDEN.

Current grammar rules:
- Function definition: use `:define name [args] ->` instead of `def name(args):`
- Return value: use `<= value` instead of `return value`
- Conditional: use `value ?cond: result ;else: alt_result ;end` instead of `if`
- Loop: use `loop var over iterable { body }` instead of `for var in iterable:`
- Block end: use `;;` to close function definition

[Task]
Write a program in this L2 language that:
1) Defines a function named fib that computes Fibonacci numbers
2) Base case: if n <= 1, return n
3) Recursive case: return fib(n-1) + fib(n-2)
4) Loop i from 0 to 6 (inclusive) and print each (i, fib(i))

IMPORTANT: Do not use any Python keywords. Use only L2 syntax as described above.
