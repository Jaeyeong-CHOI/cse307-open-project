NL-DSL 인터랙티브 데모

자연어처럼 보이는 입력을 토큰화하고, 문맥(섹션/모드/사용자 정의/seed 사전) 우선순위로 의미를 결정하는 데모입니다.

Section 우선 Mode 우선 User define Seed override Default dict

코드/로직 설명

  • 입력: NL-DSL 텍스트
  • 파싱 단계: 헤더(섹션/모드) 인식 → define token 반영 → 다의 토큰 검색
  • 의미 결정: Section > Mode > UserDefine > SeedOverride > Default
  • 출력: 토큰별 해석 로그(JSON), rule 근거 포함

참고: 저장소의 tools/nldsl_parser.py와 동일한 규칙을 웹에서 재현했습니다.

입력 제출

NL-DSL 입력

Seed override(JSON, 선택)

실행 결과

아직 실행되지 않았습니다.
