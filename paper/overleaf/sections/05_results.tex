\subsection{L1: Partial resistance with clear model spread}
All models show non-zero KLR (0.26--0.69). Context-pack halves prior influence for gpt-4o-mini. Model size does not predict compliance: gpt-5.4-mini (KLR=0.670) is worse than gpt-4o (KLR=0.366).

\subsection{L2: Syntax inversion not resistant}
gpt-4o-mini (and others tested) adopted L2 syntax fully when given explicit rules (SIR=0.0, L2-adoption=1.0). LLMs can follow explicit structural rules.

\subsection{L3: Model capability threshold}
With explicit semantic inversion rule: gpt-4o-mini fails completely (PPR=1.0, uses Python \texttt{if n<=1: return n} pattern without inversion). gpt-4o and gpt-4.1-mini succeed (PPR=0.0). This reveals a capability threshold for following semantic meta-rules.

\subsection{L4: Pattern blindness in all tested models}
Without explicit rule: gpt-4o completely ignores the semantic inversion in examples and produces standard Python fib. Pattern blindness occurs even in a strong model that can follow explicit L3 rules.

\subsection{Two Failure Modes}
\textbf{Prior Dominance} (L1, L3-weak-models): Model processes the instruction but pretrained token distribution overrides compliance.
\textbf{Pattern Blindness} (L4): Model does not extract the semantic rule from examples; outputs fall back to dominant training patterns without even attempting rule adherence.
