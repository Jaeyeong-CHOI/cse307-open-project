\section{Prompt and Evaluation Protocol}

\subsection{L1: Token Alias Substitution}
L1 system prompts instruct the model to follow alias mappings strictly and return only code. The context-pack variant adds an explicit alias contract. User prompts specify 6 keyword aliases (if, def, for, in, return, print) with varied mappings across v1--v120.

\subsection{L2: Syntax Inversion}
The model receives a complete grammar specification replacing Python constructs with novel syntax (\texttt{:define name [args] ->}, \texttt{<= value}, ternary \texttt{?cond:}, \texttt{loop var over}). All Python keywords are explicitly forbidden.

\subsection{L3: Semantic Inversion (Explicit)}
System prompt states: ``if-blocks run when condition is FALSE.'' User prompt provides one worked example and the Fibonacci task.

\subsection{L4: Semantic Inversion (Implicit)}

\begin{figure}[!t]
\small
\begin{tcolorbox}[colback=purple!4, colframe=purple!55,
  title=\textbf{L4 user prompt (example-only delivery, no explicit rule)},
  fonttitle=\small\bfseries, boxrule=0.6pt]
Here are some examples in our programming language:\\[4pt]
\textbf{Example 1 --- max function:}
\begin{verbatim}
def max_val(a, b):
    if a > b:
        return b   % smaller when a > b
    return a
\end{verbatim}
Output: \texttt{max\_val(3,5)=3},\ \texttt{max\_val(7,2)=2}\\[4pt]
\textbf{Example 2 --- even check:}
\begin{verbatim}
def is_even(n):
    if n % 2 == 0:
        return False
    return True
\end{verbatim}
Output: \texttt{is\_even(4)=False},\ \texttt{is\_even(3)=True}\\[4pt]
\textbf{Task:} Write \texttt{fib(n)} in this language.
Target: \texttt{fib(0)=0}, \texttt{fib(1)=1}, \texttt{fib(n)=fib(n-1)+fib(n-2)}.\\
Print \texttt{fib(0)} through \texttt{fib(6)}.
\end{tcolorbox}
\caption{L4 prompt (no system message). The semantic inversion rule is not stated; models must infer it from worked examples. Even gpt-4o, which succeeds at L3, defaults to canonical Python---exhibiting \emph{pattern blindness}.}
\label{fig:l4_prompt}
\end{figure}

\subsection{Judge Protocols}
\textbf{L1:} (1)~Check alias present, original token absent; (2)~normalize aliases to Python via regex; (3)~\texttt{ast.parse()} on normalized code; (4)~verify fib skeleton.

\textbf{L3/L4:} Check whether model uses inverted comparison (\texttt{if n > 1}) vs.\ standard Python (\texttt{if n <= 1}). PPR=1 if standard; PPR=0 if inverted.
